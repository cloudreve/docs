# 命令行参数 {#cli}

## 全局参数 {#global-parameters}

| 参数                      | 说明                                                                                          |
| ------------------------- | --------------------------------------------------------------------------------------------- |
| `-c`, `--conf`            | 指定配置文件路径，默认为同级目录下的 `data/config.ini`。                                      |
| `-w`, `--use-working-dir` | 默认为 `false`, 指定是否使用当前工作目录来计算相对路径，在 `false` 时使用可执行文件所在目录。 |

## 启动 Cloudreve 服务器 {#start-cloudreve}

```bash
./cloudreve
./cloudreve server
```

| 参数                  | 说明                  |
| --------------------- | --------------------- |
| `-l`, `--license-key` | 指定 Pro 版授权密钥。 |

## 提取静态资源

```bash
./cloudreve eject
```
